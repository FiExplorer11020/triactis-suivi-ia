Partage temps réel (Firebase)

Pour que toute l'équipe suive et édite via un lien, l'app se branche sur une base Firebase gratuite. Une fois configurée, chaque changement est synchronisé en temps réel pour tout le monde.
  1. Va sur console.firebase.google.com → Ajouter un projet (gratuit).
  2. Crée une base Firestore (mode test pour démarrer).
  3. Réglages du projet → Tes applications → Web → copie l'objet firebaseConfig.
  4. Colle-le ci-dessous, puis Activer.
Astuce : pour un vrai lien public partagé, héberge ce fichier (Firebase Hosting, ou tout hébergeur statique). Demande-moi de « finaliser Firebase » et je peux intégrer la config directement dans le fichier livré.
